Health Care System

What is it

A complete health care management system tailored for a specific country.

Generic features

Each country might define a group of specific features, but the base package will contain the following features

  1. Patient Registration
  2. Staff Management
  3. Medical Supplies Management
  4. Reporting Dashboard : Diseases, drugs, staff, etc...
  5. Alerting : Outbreaks, Missing drugs, Staff abuses
